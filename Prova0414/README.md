# 🎬 Catálogo de Filmes e Séries

## 📌 Descrição

Este projeto é uma aplicação desenvolvida em **React** para cadastro e exibição de filmes e séries.
O usuário pode adicionar novos títulos, visualizar em formato de catálogo e filtrar por gênero.

---

## 🚀 Funcionalidades

* ➕ Adicionar filmes ou séries
* 🎭 Definir gênero e tipo (filme/série)
* 📅 Informar ano de lançamento
* ⭐ Adicionar nota
* 🔍 Filtrar por gênero
* 🗑️ Remover itens do catálogo

---

## 🛠️ Tecnologias utilizadas

* React (useState)
* JavaScript
* HTML5
* CSS3

---

## 📸 Interface

A aplicação possui:

* Barra superior (StatusBar)
* Formulário em cards para cadastro
* Catálogo de filmes em formato de grid
* Footer com informações do autor
